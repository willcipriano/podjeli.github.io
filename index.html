Podje.li is a web based file sharing platform with a twist. In order to avoid the headache and cost of storing all those files (have you seen S3 outbound prices these days?), podje.li encodes a file into one or more URLS that you can share. This works great for small text files as they become one or two urls that can reasonably be shared, on the other hand larger files can be a problem... Check out my blog post about it!
